# ttt-calc

Simple calculator written in python, qtquick and pyqt5


## Environment

Ubuntu 64bit 20.04

### Prerequisites

```
python3
python3-pyqt5.qtquick
qml-module-qtquick-controls
pyqt5-dev-tools
qtquickcontrols2-5-dev
qml-module-qtquick2
qml-module-qtquick-layouts
qml-module-qtquick-controls2
```
